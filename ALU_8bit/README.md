# 8-bit ALU (Verilog)

## Description
Designed an 8-bit Arithmetic Logic Unit using Verilog HDL to perform various arithmetic and logical operations.

## Features
- Addition and Subtraction
- AND, OR, XOR operations
- Bitwise NOT
- Left and Right Shift operations
- Combinational logic design

## Tools Used
- Xilinx Vivado
- Verilog HDL

## Files
- alu_8bit.v (Design)
- alu_tb.v (Testbench)

## Result
Successfully simulated and verified using waveform analysis.
