# Serial Communication (Verilog)

## Description
Designed a basic serial data transmitter module to send data bit-by-bit using clock signals.

## Features
- Serial data transmission
- Bit-by-bit output
- Clock-based operation
- Sequential logic design

## Tools Used
- Xilinx Vivado
- Verilog HDL

## Files
- serial_tx.v (Design)
- serial_tb.v (Testbench)

## Result
Successfully simulated serial transmission of data.
