# Traffic Light Controller (FSM - Verilog)

## Description
Implemented a traffic light controller using Finite State Machine (FSM) in Verilog.

## Features
- Three states: Red, Green, Yellow
- Sequential state transitions
- FSM-based design
- Clock-driven operation

## Tools Used
- Xilinx Vivado
- Verilog HDL

## Files
- traffic_light.v (Design)
- traffic_tb.v (Testbench)

## Result
Successfully verified state transitions using simulation.
