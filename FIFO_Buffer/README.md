# FIFO Memory Buffer (Verilog)

## Description
Implemented a simple FIFO (First In First Out) memory buffer using Verilog for data storage and retrieval.

## Features
- Synchronous read and write operations
- FIFO behavior using memory array
- Write and Read pointer logic
- Sequential circuit design

## Tools Used
- Xilinx Vivado
- Verilog HDL

## Files
- fifo.v (Design)
- fifo_tb.v (Testbench)

## Result
Verified correct data flow using simulation.
